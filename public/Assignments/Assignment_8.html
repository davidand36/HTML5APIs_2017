HTML5 APIs Assignment 8: Templates

Create a Web app like the one for the Object List, Local Storage or REST API assignments with a table for the items and a form for creating and editing them. (You can decide which one to base this on.)

Instead of having two <div>s and togging them between hidden and shown, start with an empty <div> for your content. Create two templates, one for the table and one for the form, with embedded expressions for the data. Then use Handlebars (or another template library) to fill in the data and generate the HTML that you put into the main <div>.
